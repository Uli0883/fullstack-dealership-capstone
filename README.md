# fullstack_developer_capstone

## Proyecto Final - Cars Dealership

Este es el repositorio del proyecto final del curso de Full Stack Developer de IBM.

**Repositorio:** `xrwvm-fullstack_developer_capstone`

**Proyecto:** `fullstack_developer_capstone`

## Descripción

Aplicación web completa para la gestión de concesionarios de autos, que incluye:
- Autenticación de usuarios (login, logout, registro)
- Visualización de concesionarios
- Filtrado de concesionarios por estado
- Reseñas de concesionarios
- Análisis de sentimientos
- Panel de administración Django

## Tecnologías utilizadas

- Django (Backend)
- React (Frontend)
- Bootstrap (Estilos)
- SQLite (Base de datos local)
- Docker (Contenedores)
- IBM Cloud Code Engine (Despliegue)
